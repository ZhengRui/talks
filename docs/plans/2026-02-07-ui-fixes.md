}

### Task 6: Fix stagger animation timing for late items

Items 4+ in `anim-stagger` and `anim-counter` are nearly invisible in screenshots because their delay (`i * 120ms` or `i * 200ms`) is too long. Cap max delay.

**Files:**
- Modify: `src/styles/animations.css:91-93` (`.slide.active .anim-stagger > *`)
- Modify: `src/styles/animations.css:161-163` (`.slide.active .anim-counter > *`)

**Step 1: Reduce stagger delay multiplier**

In `animations.css`, change stagger delay from `120ms` to `100ms`:

```css
.slide.active .anim-stagger > * {
  animation: fadeUp 0.5s ease both;
  animation-delay: calc(var(--i, 0) * 100ms);
}
```

And counter delay from `200ms` to `150ms`:

```css
.slide.active .anim-counter > * {
  animation: countUp 0.7s ease both;
  animation-delay: calc(var(--i, 0) * 150ms);
}
```

**Step 2: Verify — 4th stat card and 4th+ grid items appear faster**

**Step 3: Commit**

```
fix(animations): reduce stagger/counter delay for faster item reveal
```

### Task 10: Redesign homepage with presentation cards

Replace bare bullet list with styled cards.

**Files:**
- Modify: `src/app/page.tsx` (full rewrite)

**Step 1: Rewrite homepage with cards**

```tsx
import Link from "next/link";
import { discoverPresentations } from "@/lib/loadPresentation";

export default function Home() {
  const presentations = discoverPresentations();

  return (
    <main className="min-h-screen bg-[#0a0a12] text-white p-8 md:p-16">
      <div className="max-w-4xl mx-auto">
        <h1 className="text-5xl font-bold mb-2">Presentations</h1>
        <p className="text-gray-400 mb-12">Select a presentation to begin</p>
        {presentations.length === 0 ? (
          <p className="text-gray-500">No presentations found.</p>
        ) : (
          <div className="grid gap-4">
            {presentations.map((p) => (
              <Link
                key={p.slug}
                href={`/${p.slug}`}
                className="group block p-6 rounded-xl border border-white/10 bg-white/5 hover:bg-white/10 hover:border-white/20 transition-all"
              >
                <div className="flex items-center justify-between">
                  <div>
                    <h2 className="text-xl font-semibold group-hover:text-blue-400 transition-colors">
                      {p.title}
                    </h2>
                    {p.author && (
                      <p className="text-gray-400 text-sm mt-1">by {p.author}</p>
                    )}
                  </div>
                  <span className="text-gray-500 text-sm whitespace-nowrap ml-4">
                    {p.slideCount} slides
                  </span>
                </div>
              </Link>
            ))}
          </div>
        )}
      </div>
    </main>
  );
}
```
